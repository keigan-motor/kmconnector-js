cmdMaxTorque(maxTorque)

  • KMMotorCommandKMOne.js, line 912
Inherited From:
  • KMMotorCommandKMOne#cmdMaxTorque

maxTorque を設定することにより、トルクの絶対値が maxTorque を超えないように運転します。
